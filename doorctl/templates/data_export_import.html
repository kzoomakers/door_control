Export Data

Export all database data to a JSON file. This includes:

  • User/Card information
  • Event logs (optional)
Note: The exported file can be used for backup purposes or to migrate data to another system.

Import Data

Import data from a JSON file exported from this system.

Warning: Importing data can modify your database. Make sure to backup your current data before importing.
Merge: Adds new records without affecting existing data.
Replace: Deletes all existing data before importing.

API Access

You can also export and import data programmatically using the REST API:

Export via API
GET /api/v1/export
Headers:
  X-API-Key: your-api-key

Query Parameters:
  include_events=true/false  (default: true)
  download=true/false        (default: false)
Import via API
POST /api/v1/import
Headers:
  X-API-Key: your-api-key
  Content-Type: application/json

Body:
{
  "data": {
    "users": [...],
    "events": [...]
  },
  "mode": "merge",           // or "replace"
  "skip_duplicates": true
}
